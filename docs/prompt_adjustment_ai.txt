> ### 1. Nova seção: Métricas de Dados
>
> Crie um menu dentro de:
> **Conteúdo & Vendas → Métricas de Dados**
>
> Essa seção deve conter os seguintes campos e indicadores:
>
> * Análise de seguidores
> * Cliques
> * Engajamento
> * Quantidade de campanhas
> * Alcance (seguidores vs não seguidores)
>
> **Requisitos:**
>
> * Apresentar os dados de forma visual (gráficos, cards ou dashboards)
> * Garantir organização clara e leitura rápida
> * Preparar estrutura para integração futura com APIs de dados
>

> ### 3. Gestão de Usuários e Perfis
>
> Criar um módulo completo de controle de acesso com:
>
> * Cadastro de usuários
> * Perfis (Roles)

>
> **Requisitos:**
>
> * Sistema baseado em RBAC (Role-Based Access Control)
> * Possibilidade de criar, editar e excluir perfis
> * Associar permissões específicas a cada perfil
> * Criar Perfis, papeis e permissões em área dedicada a isso (acesso default somente pelo Super ADM)
